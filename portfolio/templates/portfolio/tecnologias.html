Front-End

O programador responsável pelo front-end é quem “dá vida” à interface gráfica entre o utilizador e o website, para que os utilizadores possam ver e interagir com o website. Exemplos:

HTML - Hyper Text Markup Language

Criado em 1991 por Tim Berners-Lee

É uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

CSS - Cascading Style Sheets

Criado em 1996 por Hakon Lie

É um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags style.

Back-End

Como o nome sugere, o desenvolvedor back-end trabalha na parte de “trás” da aplicação. É o responsável, em termos gerais, pela implementação das regras de negócio e não toca na parte visual da aplicação. Exemplos:

Java

Criado em 1995 numa equipa chefiada por James Gosling

Java é uma linguagem de programação orientada a objetos desenvolvida na década de 90 por uma equipe de programadores chefiada por James Gosling.

Python

Criado em 1989 numa equipa chefiada por Guido Van Rossum

Python é uma linguagem de programação de alto nível, interpretada de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte.

C Sharp (C#)

Criado em 2000 pela Microsoft

C# é uma linguagem de programação, multiparadigma, de tipagem forte, desenvolvida pela Microsoft como parte da plataforma .NET.
